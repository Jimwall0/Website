# Personal Portfolio Website

This is a personal portfolio website for **Tsim Muaj Yang**, built using basic **HTML and CSS**. The site introduces who I am, highlights my background in tech, showcases some of my projects, and provides ways to contact me.

## Features

- About Me section with a short personal introduction  
- Portfolio section showcasing school and personal projects  
- Contact section with links to GitHub, LinkedIn, resume, and email  
- Responsive layout using basic HTML and CSS  

## Technologies Used

- CSS3
